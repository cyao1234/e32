# ESP32-s2 development board

With drunk components!

---

![](https://files.catbox.moe/6rd4px.png)

This is a single board esp32-s2 development board. with two neopixels and an on-board antenna.

It has drunk component placement and some nice freeform traces

The 2.54mm pitch can be fitten in breadboard and used nicely with jumpers

The programming can be done over USB-C ports

---

![](https://cdn.hackclub.com/019e7eb6-f2ad-717b-9d17-6b37124bbe63/image.png)

The scehmatic is easy and simple, giving people the maximum leverage.

Everything is impedance matched, giving the best EMC and high-speed SI performance
